# Руководство по работе с платежным API через Swagger

Данное руководство поможет разработчикам быстро протестировать отправку платежей в системе **TechWriter.PRO**.

## Перед началом работы
- Получите токен авторизации у вашего менеджера.
- Откройте тестовый стенд Swagger по ссылке: [Интерфейс Swagger](https://swagger.techwriter.pro)

## Тестирование метода /pay (Создание платежа)

1. Перейдите в раздел **`Payment-Controller`**.
2. Найдите POST-запрос по адресу `/api/v1/pay`.
3. Нажмите кнопку **Try it out** (Начать тестирование).
4. Вставьте в поле Тело запроса (Request Body) следующий JSON-код:

```json
{
  "amount": 5000,
  "currency": "RUB",
  "order_id": "SPARTAK-2026"
}
